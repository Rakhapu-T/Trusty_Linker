## 1. Installation

- Use AppImage or a full access installation of your system's package manager instead. On Ubuntu, download the [.deb](https://github.com/obsidianmd/obsidian-releases/releases/download/v1.11.7/obsidian_1.11.7_amd64.deb) package. This is for version control and syncing using the **GIT** plugin.

## 2. Theme
Either one of the following themes:
- Obsidianite
- Things
